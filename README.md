# 🚀 GraphQL E-commerce API

Projet académique : API GraphQL pour un système e-commerce avec Spring Boot et React.

---

## 🧠 Stack technique

- Backend : Spring Boot + GraphQL
- Frontend : React + Apollo Client
- Database : PostgreSQL
- Auth : JWT / Keycloak
- Tests : Postman
- Temps réel : WebSocket Subscriptions
